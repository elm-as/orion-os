## 🎯 VISION
**Simulation d'OS web pour la visualisation télémétrique de serveurs.**

Une interface immersive traitant des flux de données simulés. Ce dashboard explore des paradigmes d'UX système pour monitorer des états distribués, utilisant les WebSockets pour une synchronisation à faible latence.
